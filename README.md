## Requirements

Base R and the RStudio IDE.

## Reproducing Results

In the RStudio IDE, open the file _messinian_biodiv.Rproj_. This opens the RProject of the same name. Then, run

```R
source("code/analysis.R")
```

to produce all figures in the _figs_ folder.
